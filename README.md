# AkilliEvIOT
Projemiz proteus üzerine çizilmiş ve ardunioC kullarak yazılmıştır
Kullanılan sensorler yangın için flamesensor, sıcaklık için lm35, hareket algılama için PIRsensor ve şifre girişi için 3*4 keypad
Yangın sensorümze 0 veya 1 inputları verilerek buzzerın çalması sağlanmaktadır.
PIR hareket sensorumuz ile 0 veya 1 inputları verilerek Lambanın yanmasını sağlar.
Keypadimizde belirlenen şifre dışında sifre girilse kırmızı, doğru şifre ise yeşil ışık yanar.
Lm35 ile sıcaklık hesaplanması yapılır ve LCD üzerine gerekli bilgiler gosterilmektedir.
